Usage

The <go-button> component should be used for any instance of a button throughout the app.

Bindings
buttonDisabled

Disables the button. This should be handled by the parent component.

buttonIcon

Add an icon to the left of the text in the button. See the Material Design System Icons page for available icons.

buttonType

Changes the appearance of the button. There are two types, default and alert. You only need to set this binding if a type other than default is desired.

useLoader

Allows the use of the loading animation inside the button. By default this is false. It is recommended to use this when processing data that will take longer than 250ms. Examples: submitting a form, sorting data, refreshing data from the server, etc.

handleClick

This event fires when the button is clicked. Use this event to handle the click event.
