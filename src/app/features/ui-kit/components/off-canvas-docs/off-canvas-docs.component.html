Off Canvas

Usage

The <go-off-canvas> component should be the only instance of a modal in the app.

Setup

To use the off canvas anywhere in the app, you must first add it to the app level. This should only be done once per application.

  1. Import the GoOffCanvasModule into the main app module.

  2. Add the <go-off-canvas> element to app.component.html.

Using the Off Canvas service

To make use of the off canvas in our component:

  1. In the module that imports your component, import the GoOffCanvasModule, add the component that will be rendered in the modal to the entryComponents, and add the GoModalService to the providers.

  2. In the component that is telling the modal to open, import the GoModalService, and the component that is being rendered inside the modal.

  3. To open the off canvas, use the service to call openOffCanvas. This method takes the component type (BasicTestComponent in our case), and the bindings for that component as an object.

  4. Lastly, we need something to trigger that function. In our case we will use a button that triggers the openOffCanvas method, but it could be triggered by any event.
